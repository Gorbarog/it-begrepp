# Mattias IT-handbok (statisk webb)

Detta är en lätt, robust och helt statisk webbplats (HTML/CSS/JS) som passar utmärkt för att experimentera med web hosting.

## Struktur
- `index.html` – startsida
- `azure.html` – Azure-anteckningar
- `natverk.html` – nätverk (DNS/DHCP)
- `labb.html` – labb-upplägg
- `checklistor.html` – checklistor (med utskriftsläge)
- `kontakt.html` – kontakt/förvaltning
- `assets/` – CSS, JS, ikoner

## Publicering
Ladda upp innehållet i valfri webbhosting. För GitHub Pages ska `index.html` ligga i repo-roten.

## Tips
Kopiera en sida och ändra titel + innehåll när du vill skapa en ny sektion.
